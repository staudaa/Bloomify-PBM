# Bloomify

Bloomify adalah aplikasi Flutter katalog bunga yang terintegrasi dengan REST API.

# Features

- Login Authentication
- API Integration
- Product Catalog
- Add Draft Product
- Submit Product Task
- Soft Delete Product
- Beautiful Soft Pink & Lavender UI

# Technologies

- Flutter
- REST API
- HTTP Package
- Flutter Secure Storage
